Este proyecto le da la opción al usuario de trabajar con arboles AVL con la funcionalidad de ir insertando los valores del arbol de manera manual o directamente desde
un archivo csv. También el programa va acomodando el arbol con cada insercion nueva que haga el usuario al arbol respetando el balance de -1 a 1 para mantener equilibrado
el arbol. Para insertar los csv para crear los arboles el programa le solicitara al usuario que ingrese directamente la ruta de acceso donde esta almacenado el archivo,
adicionalmente vale la pena mencionar que el programa solo reconoce la ruta sin las comillas ("") que agrega la ruta de accesso cuando se copia directamente del 
explorador de archivos si se llegase a ingresar la ruta con las comillas el programa no lo reconocera y no podra agregar los datos deseados. El programa da la alternativa
de insertar, eliminar y buscar elementos dentro del arbol. Tambien puede generar representaciones graficas del arbol por medio de gravipvhz, solo que hay que pedirle al
programa que vaya generando las imagenes, el no lo hace automatico cada que se ingresa un nuevo dato, es decir, si ingresa los datos por csv debe pedirle al programa 
que genere la imagen, si inserta o elimina un dato tiene que pedirle nuevamente que lo genere para obtener una imagen con los datos actualizados del arbol. Por cuestiones
de funcionalidad para calificacion, la ruta donde se guarda la imagen generada del arbol es generica para guardarse en el usuario de la computadora que esta usando, es 
decir, si gusta buscar la imagen tienen que dirigirse al explorador de archivos y buscar la imagen con el nombre que ingreso en el programa para guardar la imagen. asi
podra ver el arbol AVL.
