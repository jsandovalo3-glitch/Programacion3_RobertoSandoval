Esta aplicacion sirve para generar un arbol B desde un archivo .csv, permitiendo que el grado del arbol sea perzonalizable a la eleccion del usuario y tambien generando una imagen utilizando graphizv para representar de manera visual el arbol y que el usuario lo pueda visualizar. Para insertar los datos desde un archivo .csv tiene que elegir la opcion 4 del menu de la aplicacion y debe de ingresar la ruta donde usted tiene guardado el archivo, excluyendo las comillas; por ejemplo la ruta "C:\Users\ovall\Documents\Mariano Galvez\Primer semestre 2026\Progra III\ARCHIVO3_20ELEMENTOS.csv" debe de escribirla sin las comillas. Luego le da la opcion de mostrar el arbol en consola con los niveles del arbol y tambien le da la oportunidad de generar una imagen de tipo .png para la representacion visual del arbol. Tambien puede buscar claves dentro del arbol, insertar y eliminar claves haciendo las adaptaciones necesarias para no romper el equilibrio del arbol.
