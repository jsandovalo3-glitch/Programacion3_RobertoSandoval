Tarea No.2 Recursividad 

En este repositorio se encuentra el codigo fuente de la tarea no.2, donde se presenta una aplicacion que hace uso del tema aprendido de la recursividad. El programa
cuenta con 5 opciones para el usuario:

1. Convertir numero entero a binario
2. Contar digitos de un numero entero
3. Calcular raiz cuadrada de un entero positivo
4. Suma de enteros hasta un numero ingresado
0. Salir y cerrar programa

Para elegir una de las opciones con las que se desea trabajar, el usuario debe de ingresar unicamente el numero al inicio de cada opcion. Por ejemplo, si el usuario desea
convertir un entero a binario tiene que escribir unicamente el numero 1 para luego ser dirijido al siguiente menu donde le solicitara los datoe para convertir el numero
con el que desea trabajar. 

El funcionamiento de la primera opcion el programa le solicita al usuario que escriba un numero entero (unicamente positivo) para luego regresar el valor de ese entero 
representado como un numero binario. Para la segunda opcion de igual manera se solicita un entero positivo para luego regresarle al usuario la cantidad de digitos que 
tiene el numero que ingreso. La opcion 3 calcula la raiz cuadrada unicamente de enteros positivos, la aplicacion de igual manera le solicita al usuario que ingrese un 
entero para luego regresar el resultado de la raiz cuadra. La cuarta opcion realiza una suma de enteros hasta el numero que ingrese el usuario, por ejemplo, si el usario
ingresa el numero "5" el programa le regresa la suma de enteros hasta ese numero (0+1+2+3+4+5) por lo que le estaria regresando el numero "15". Por ultimo se presenta 
la opcion de salir que como lo dice la instruccion siver para salir y finalizar el programa. 

Adicionalmente, no se pudo implementar la cuarta funcion recursiva que se solicitaba que era la de convertir un numero romano a entero.
